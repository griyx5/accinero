# VaccinePro — Автоматизированная система учёта вакцинации и медицинских процедур

## Описание
Приложение для ведения электронного учёта вакцинаций и медицинских процедур пациентов. Разработано в рамках учебной практики.

## Функциональность
- Регистрация и управление карточками пациентов (ФИО, дата рождения, телефон, СНИЛС)
- Внесение записей о вакцинациях (препарат, доза, серия, дата)
- Внесение записей о медицинских процедурах (название, врач, диагноз, дата)
- Просмотр истории по каждому пациенту
- Автоматический расчёт даты следующей ревакцинации
- Отчёт по пациентам, которым требуется ревакцинация
- Авторизация с разграничением ролей (админ, медсестра, врач)

## Технологии
- Python 3.10+
- Tkinter (GUI)
- SQLAlchemy (ORM)
- SQLite (база данных)
- PyInstaller (сборка .exe)
- Git (система контроля версий)
