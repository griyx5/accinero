# Sevilia - информационная панель Python Community

## Запуск
1. Скачайте `Sevilia.exe` и `sevilia.db` в одну папку.
2. Запустите `Sevilia.exe`.
3. Логин: `admin`, пароль: `admin123`.

## Функции
- Просмотр статистики (онлайн, загрузки, файлы).
- Управление списком версий Python (CRUD).

## Возможные проблемы
- Если таблицы не созданы – удалите `sevilia.db` и перезапустите.
- При ошибке входа проверьте, что рядом с .exe есть файл базы.
